- ✅ **100+ funktionierende Tweaks** für Windows 10/11

- 🔒 **Privacy-Protection** - Telemetrie & Tracking deaktivieren
- 🗑️ **Debloat** - Bloatware & vorinstallierte Apps entfernen

# Abhängigkeiten installieren
